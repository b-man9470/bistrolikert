# bistrolikert

Dit is de website van bistro likert, een restaurant gelegen in Liedekerke, België.

#Benodigdheden
* De menu moet duidelijk weergegeven worden op de website
* De contactgegevens moeten aanwezig zijn op de website
* Het moet mogelijk zijn voor de eigenaar om de menu gebruiksvriendelijk aan te passen (enkel de eigenaar mag dit kunnen doen)
* De suggestie van de week moet duidelijk zichtbaar zijn voor de klanten
* De eignaar moet gebruiksvriendelijk de suggestie kunnen posten eventueel met een foto
